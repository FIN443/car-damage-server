# Car damage detection server

Car damage part detection server with YOLO custom.

## Model

- YOLO v4
- Traning car images: 128244
- min_val_loss: 316

## Requirements

```
tensorflow==2.8.1
numpy==1.21.5
flask==2.1.2
flask_cors==3.0.10
opencv-python==4.6.0.66
```

## How to run?

```
python main.py
```
